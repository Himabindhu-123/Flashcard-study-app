# 📚 Flashcard Study App

An interactive and responsive **Flashcard Study Web Application** built using **HTML, CSS, and JavaScript**.  
The app helps users study by flipping flashcards, tracking progress, and managing their own study cards.


## 🌐 Live Demo

https://himaflashcard.netlify.app/


## ✨ Features

- **Flashcard Flip Animation** – Click the card to reveal the answer  
- **Add Flashcards** – Create your own study cards  
- **Delete Flashcards** – Remove unwanted cards  
- **Shuffle Cards** – Randomize flashcard order  
- **Progress Bar** – Track study progress  
- **Card Counter** – Displays the current card number  
- **Local Storage Support** – Saves flashcards even after page refresh using the browser Local Storage API  
- **Celebration Effect** – Confetti animation when all flashcards are completed using the Canvas Confetti library  
- **Responsive Design** – Works on both mobile and desktop devices  


## 🛠 Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**
- **Font Awesome Icons**
- **Local Storage API**


## 🚀 How to Run the Project

1. Clone the repository

2. Open the project folder.

3. Run **index.html** in your browser.


## 📖 How to Use

1. Click the flashcard to flip and reveal the answer.  
2. Use **Next** and **Previous** buttons to navigate cards.  
3. Click **Shuffle** to randomize flashcards.  
4. Add your own flashcards using the input fields.  
5. Delete cards if needed.  
6. Complete all flashcards to see the celebration effect.
